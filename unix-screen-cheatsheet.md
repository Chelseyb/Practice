# Unix Screen Cheatsheet

## Basic commands

| Task | Command|
| --- |:---|
| Start a new screen session with session name  | screen -S [name] |
| List running sessions/screens  | screen -ls |
| attach to a running session | screen -x |
| attach to session with name  | screen -r [name] |

## Multi user mode

```
^a:multiuser on
```
